# Тестирование приложения Random Task Generator

**Проект:** Random Task Generator  
**Автор:** Алексей Смирнов  
**Назначение тестирования:** проверка корректной работы GUI-приложения для генерации случайных задач.

---

## 1. Что проверялось

В ходе тестирования были проверены основные функции приложения:

- запуск GUI-приложения;
- генерация случайной задачи;
- фильтрация задач по типам;
- добавление новой задачи;
- проверка корректности ввода;
- сохранение данных в JSON;
- загрузка данных из JSON;
- отображение истории генераций;
- очистка истории;
- отображение статистики.

## 5. Проверка JSON

Пример структуры файла `tasks.json`:

```json
{
  "tasks": [
    {
      "text": "Прочитать статью по Python",
      "type": "учёба",
      "date": null
    }
  ],
  "history": [
    {
      "text": "Сделать 30 минут зарядки",
      "type": "спорт",
      "date": "11:20:05"
    }
  ]
}
```

Проверено:

- данные сохраняются в JSON;
- русские символы отображаются корректно;
- история сохраняется после генерации задачи;
- после перезапуска программы история загружается обратно;
- при очистке истории JSON-файл обновляется.
