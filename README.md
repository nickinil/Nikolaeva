**Автор:** Николаева Аэлита Дмитриевна

**Дата:** 2026

## Описание

Random Task Generator - это десктопное GUI-приложение для генерации случайных задач с возможностью фильтрации по типу, ведения истории и сохранения данных. Приложение помогает разнообразить рутину и случайным образом выбирать задачи для учёбы, спорта или работы.

## Функциональность

- ✅ Генерация случайной задачи из списка
- ✅ Фильтрация по типам: учёба, спорт, работа
- ✅ Отображение истории сгенерированных задач с временем
- ✅ Добавление новых задач с проверкой ввода
- ✅ Сохранение и загрузка данных в/из JSON
- ✅ Очистка истории
- ✅ Статистика генераций

## Требования

- Python 3.6 или выше
- Операционная система: Windows / macOS / Linux
